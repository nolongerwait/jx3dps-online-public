## 装备数据结构

```
dwTabIndex: 装备ID
nPos: 装备部位
nStrengthLevel: 精炼等级
dwPermanentEnchantID: 附魔 ID
dwItemFEAEnchantID: 五彩石 ID
dwTemporaryEnchantID: 大附魔 ID
aDiamondEnchant: 镶嵌列表
```
